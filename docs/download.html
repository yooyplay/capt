Installation steps

1

Open the DMG and drag to Applications

Double-click Capt.dmg, then drag Capt.app into the Applications folder shown in the window.

2

macOS will show a security warning

Because Capt is not on the Mac App Store, macOS blocks it on first launch:

!
"Capt" Not Opened
Apple could not verify "Capt" is free of malware that may harm your Mac or compromise your privacy.
Don't click "Move to Trash"

This happens with any app not sold through Apple's store. Capt is open source — nothing is hidden.

Open Terminal (Cmd Space, type Terminal, hit Enter) and run:

xattr -cr /Applications/Capt.app

Then double-click Capt normally. One time only.

3

First launch installs everything automatically

Capt installs openai-whisper, moviepy, and a bundled ffmpeg via pip on first run. Only requires Python 3:

brew install python

After first-time setup, everything runs fully offline.

4

Drop a video and go

Pick a Whisper model size, drop your video, click Caption. The captioned file is saved next to the original.

💡 Tip: Want to skip the security warning entirely? Install via Homebrew instead.

🍺 What is Homebrew?

Homebrew is a free package manager for Mac. You run a command in Terminal and it handles the download, checksum check, and install. No popups, no Gatekeeper warnings, and updating later is one command.

Install via Homebrew

1

Install Homebrew (skip if you already have it)

Open Terminal (Cmd Space → type Terminal) and run:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2

Install Capt

brew tap yooyplay/tap
brew trust yooyplay/tap
brew reinstall --cask capt
xattr -cr /Applications/Capt.app
open /Applications/Capt.app

Homebrew downloads and installs it with no security warnings.

3

Update anytime with one command

brew upgrade --cask capt
🔒
Runs locally — your videos never leave your Mac. Whisper transcribes everything on-device.
