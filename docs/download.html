Installation

1

Open the DMG and drag to Applications

Double-click Capt.dmg, then drag Capt.app into the Applications folder.

2

Bypass the macOS security warning

macOS blocks apps not on the App Store on first launch. Open Terminal (Cmd Space, type Terminal) and run:

xattr -cr /Applications/Capt.app

Then double-click Capt normally. One time only.

3

First launch installs Whisper automatically

Capt installs openai-whisper and moviepy via pip on first run. Needs Python 3 installed (brew install python if you don't have it). After that it's fully offline.

4

Drop a video and go

Pick a Whisper model size, drop your video, click Caption. The captioned file is saved next to the original.
